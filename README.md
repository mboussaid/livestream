# Livestreaming

Get started with building a saas product much faster by using this boilerplate code. Covers login, register, forgot password, and responsive navigation.

## Tech/framework used

Frontend tech stack:

- React JS
- CSS
- React Router
- React Icons

Backend tech stack:
- Node JS
- Express

## Installation

To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
